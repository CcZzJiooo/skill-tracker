## One-line Pitch

Skill Tracker is a local-first dashboard that visualizes AI-agent skill calls, translates skill purposes into Chinese, and helps clean up duplicated skills.

## Short Pitch

Most AI coding agents can call skills, plugins, and local workflows, but users cannot easily see what was actually used. Skill Tracker scans local session logs and turns `SKILL.md` reads into a private dashboard with heatmaps, timelines, Chinese skill descriptions, duplicate-skill governance, GitHub discovery, and exportable action plans.

## V2EX / 掘金 / 知乎 Draft

```text
我开源了一个 AI Agent 技能调用可视化工具：Skill Tracker。

它解决的问题是：现在很多 Agent 都支持 skills / plugins / prompts，但用户很难知道一次会话里到底调用了哪些 skill、哪些 skill 重复、哪些缺少中文说明。

- 技能调用热力图和时间线
- 每个 skill 的中文功能说明
- 根据中文意图搜索 skill
- 重复/相似 skill 检测
- GitHub 开源项目搜索
- 治理问题导出成 GitHub issue 草稿
- 匿名化报告导出，避免泄露本地路径和 session

项目地址：
https://github.com/CcZzJiooo/skill-tracker
```

# Skill Tracker Roadmap

This roadmap prioritizes practical adoption first, then deeper agent-skill governance.

## Phase 1: Adoption and Trust

- Clear English-first README with Chinese support.
- GitHub topics, screenshots, and release notes.
- Issue templates for bug reports, feature requests, and tool adapters.
- Privacy notes for generated telemetry files.
- Citation and attribution files.

## Phase 2: Adapter Coverage

- More skill-root auto-detection:
  - Codex skills.
  - Claude skills.
  - Gemini skills.
  - Agent skill folders.
- More session-log adapters:
  - Additional editor plugins.
  - Custom JSONL schemas.
  - Optional imported anonymized reports.

## Phase 3: Governance Intelligence

- Stronger duplicate-skill similarity scoring.
- Better conflict detection across overlapping skills.
- Trend snapshots for skill usage over time.
- Recommended cleanup workflow with dry-run reports.
- Validation for imported `skill_catalog.json`.

## Phase 4: Distribution

- CLI wrapper:
  - `skill-tracker collect`
  - `skill-tracker open`
  - `skill-tracker export`
- Optional installer.
- Cross-platform shell support.
- Hosted static demo page using synthetic data.

## Phase 5: Research and Ecosystem

- Anonymous community benchmark format.
- Skill taxonomy proposal.
- Public examples of AI-agent skill observability.
- Integrations with issue trackers and documentation sites.

# Skill Tracker Promotion Pack

这个目录放发布传播素材，目标是让同一个开源项目在不同平台用不同角度发出去，而不是复制同一篇硬广。

## 文件

- `PLATFORM_COPY.md`: 各平台可直接发布或稍改的文案。
- `assets/`: 每个平台对应的配图 PNG，由 `scripts/generate-promotion-assets.py` 生成。

## 推荐发布顺序

1. CSDN: 技术教程和 SEO。
2. 掘金: 中文开发者首发讨论。
3. OSCHINA: 开源项目扩散。
4. 知乎: 问题意识和产品动机。
5. SegmentFault / V2EX: 精准技术圈。
6. Gitee / GitCode: 国内镜像和下载入口。
7. 小黑盒: 效率工具/折腾向图文。
8. B 站 / 小红书: 用视频或图文做二次传播。

## 发布原则

- 不要说“革命性”“首创”这类不可验证词。
- 强调本地优先、隐私、可视化、中文 skill 说明、重复治理。
- 每个平台开头要像原生内容，不要像广告。
- 能放图就放 `assets/` 下对应封面。
- 发帖后把链接补回 README 或 GitHub Release 说明里，形成回流。

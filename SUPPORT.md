# Support

Use GitHub Issues for support.

## Before Opening an Issue

Check:

- You are on Windows or can explain your platform.
- `config.json` points to the intended skill root if auto-detection fails.
- Generated telemetry files are not committed.
- You can reproduce the problem with demo data or sanitized data.

## Useful Issue Types

- Bug report: something breaks or shows incorrect data.
- Tool adapter request: a new AI coding tool should be supported.
- Skill dictionary request: better Chinese descriptions or categories.
- Governance rule request: duplicate or conflict detection can be improved.
- Documentation issue: README/manual is unclear.

## Privacy Reminder

Do not paste raw local logs into public issues. Use anonymous export or redact:

- Session IDs.
- Local filesystem paths.
- Internal project names.
- Private skill names.
- Full private skill descriptions.
